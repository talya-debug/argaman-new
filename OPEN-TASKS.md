# משימות פתוחות — ארגמן

## 1. ליד = כרטיס לקוח (שינוי מהותי)
- הסרת סטטוסים מלידים
- סימון דורש טיפול / לא דורש טיפול
- לוג פעילות (מי עשה מה ומתי)
- כפתור "הצעה חדשה" + רשימת הצעות קיימות
- סטטוס: ממתין לאישור

## 2. ארכיון מרכזי
- דף ארכיון אחד (כבר נבנה) עם קטגוריות: פרויקטים, הצעות, לידים
- כפתור ארכיון בפרויקט דרך סטטוס (כבר נבנה)
- כפתור ארכיון בהצעות (כבר חלקי)
- כפתור ארכיון בלידים
- סטטוס: חלקי — צריך להשלים

## 3. בוט וואטסאפ
- ממתין לניתוח שיחות הקבוצה
- אפיון משימות מהקבוצה ("משימה:" + קבצים)
- תזכורות (יומן עבודה, חשבונות, רכבים)
- צריך שרת (Railway ~$5/חודש)
- סטטוס: ממתין לאפיון

## 4. מסך מוחשך
- נוספו CSS fixes
- סטטוס: לבדוק אם עדיין קורה
