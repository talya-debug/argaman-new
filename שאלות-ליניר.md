# יניר — דברים שצריך ממך

## 1. קריאות שירות — FIXUS
בדקנו ול-FIXUS יש API שמאפשר חיבור למערכות חיצוניות.
אנחנו יכולים למשוך את קריאות השירות שלכם אוטומטית לתוך המערכת החדשה.

מה צריך ממך:
- תתקשר ל-FIXUS (052-8928581) ותגיד להם שאתם רוצים לחבר API למערכת ניהול חיצונית
- תבקש מהם: תיעוד API + מפתח גישה (API key)
- תעביר לנו את מה שהם שולחים

## 2. הצעת מחיר לפרטיים
הבנו שאתה רוצה שהצעה ללקוח פרטי תצא עם מסמכים נוספים — אודות החברה והסבר על מערכת המיזוג.

תשלח לנו:
- קובץ PDF או וורד של "אודות החברה" אם יש לכם (אם אין — נבנה, רק תגיד מה לכתוב)
- קובץ ה"הסבר על המערכת" שאתה רוצה לצרף

## 3. רכבים — דיווח ק"מ
כרגע המערכת מאפשרת להזין ק"מ ידנית.
שאלה: מי מזין את הק"מ? חיה פעם בשבוע, או שכל נהג ידווח בעצמו דרך הטלפון?

## 4. דוחות כספיים
אמרת שתשלח קבצים — מחכים לקבל אותם.

## 5. הרשאות משתמשים
כרגע כל המשתמשים רואים הכל. אם אתה רוצה להגביל — תגיד לנו מה עובד רגיל לא צריך לראות.
למשל: מחירים? גבייה? דוחות? הצעות מחיר?

## 6. וואטסאפ
הבוט מוכן. צריך רק שתשלח לנו מספר טלפון שנחבר אליו (יכול להיות מספר רגיל, לא חייב עסקי).

## 7. משימות אוטומטיות — בדוק אם רלוונטי
המערכת יוצרת משימות אוטומטיות במקרים הבאים. תעבור ותגיד אם כולן רלוונטיות, אם צריך לשנות את מי שמשוייך אליהן, או אם יש משהו מיותר:

### כשליד מאושר ונפתח פרויקט:
- "אדמיניסטרציה – פתיחת תיקייה, כניסה למעקב גבייה" → משוייך ל: אדמיניסטרציה
- "פגישת התנעה יניר ויהודה" → משוייך ל: חיה

### כשנוצרת הזמנת רכש:
- "הזמנת רכש: [שם פריט]" → משוייך ל: האחראי שנבחר בהזמנה (ברירת מחדל: חיה)
- כולל תיאור: כמה יחידות, לאיזה פרויקט, שם ספק

### כשמזינים חשבונית ספק ברכש:
- נוצרת רשומת הוצאה אוטומטית בגבייה עם סכום שלילי (הוצאה)

### כשמאשרים חשבון חלקי בכתב כמויות:
- נוצרת משימת גבייה אוטומטית → משוייך ל: רבקה
- סטטוס: "חשבון מאושר – יש לשלוח חשבון עסקה"

### כשפרויקט עובר ל"הושלם":
- "סגירת פרויקט - [שם]" → משוייך ל: אדמיניסטרציה
- "בדיקת חשבון סופי - [שם]" → משוייך ל: דבורה
- אם מוגדר עיקבון: "עיקבון לגבייה - [שם]" → משוייך ל: רבקה, עם תאריך יעד לפי תקופת העיקבון

### כשמסמנים תשלום חלקי בגבייה:
- נוצרת משימת המשך אוטומטית על היתרה שנותרה

### שאלות:
- האם כל המשימות האלה רלוונטיות?
- האם השיוך נכון? (למשל: דבורה אחראית על חשבון סופי? רבקה על גבייה?)
- האם יש משימות שחסרות?
